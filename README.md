🧵 Hilograma

Sistema de gestión de asistencias y personal para talleres de costura

📌 Descripción

Hilograma es una aplicación diseñada para optimizar el control de asistencia y la gestión de empleados en entornos de producción textil. Combina precisión administrativa con una interfaz intuitiva, permitiendo mantener un seguimiento claro, ordenado y eficiente del personal.

🚀 Funcionalidades principales
👥 Gestión de empleados
Alta, edición y administración completa del personal
Visualización detallada por empleado
Asociación de cargos con visualización clara (nombre + cantidad)
Control de estado: activo, en nómina o dado de baja
Exclusión selectiva de empleados en procesos de nómina
Generación automática de nóminas en formato PDF y Excel

📅 Control de asistencias
Registro individual de asistencia
Gestión por rangos de fechas
Edición de registros con persistencia correcta en base de datos
Validaciones lógicas en estados (ej: control de subestados en ausencias)
Visualización de observaciones por registro
Historial centralizado de asistencias
Generación de reportes

⚡ Carga masiva
Importación de datos por lotes
Optimización de tiempos en registro de asistencias
Corrección de referencias de base de datos en procesos automatizados

📊 Dashboard y análisis
Panel interactivo con métricas clave
Diferenciación clara entre empleados en nómina y fuera de ella
Resumen de cargos por empleado activo
Visualización optimizada de datos relevantes

🔐 Seguridad y acceso
Sistema de autenticación con validación estricta
Protección contra inyecciones SQL mediante uso de PDO
Consola avanzada restringida (“Terminal Secreto”) para gestión directa de base de datos

🎨 Interfaz y experiencia de usuario
Diseño intuitivo y adaptable
Sistema de temas con cambio dinámico
Mejora continua en visualización de componentes
Dashboard optimizado para claridad y legibilidad
Reloj en tiempo real integrado
Monitor de estado del servidor

🛠️ Tecnologías utilizadas
Backend: PHP
Base de datos: MySQL
Conectividad: PDO
Frontend: HTML, CSS, JavaScript

⚙️ Características técnicas
Arquitectura basada en persistencia segura
Manejo eficiente de datos y relaciones
Optimización de consultas y estructura de tablas
Preparado para entornos de producción

📈 Enfoque del sistema
Hilograma está pensado como una herramienta escalable, capaz de adaptarse a distintos tamaños de talleres, manteniendo siempre un equilibrio entre control administrativo, claridad visual y facilidad de uso.
